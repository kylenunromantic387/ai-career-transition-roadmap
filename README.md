## 💼 Portfolio ideas
Use the roadmap to build work that shows practical skill.

Good starter projects:
- A simple chat tool that answers common questions
- A small model that sorts text into groups
- A data clean-up project with clear before-and-after results
- A prompt set for a real task, with notes on what worked
- A basic AI workflow that shows how data moves through the system

Keep each project small. Make the goal clear. Show what problem it solves.

## 🔍 Learning path
Start with the basics:
- Learn what AI, machine learning, and LLMs do
- Learn the main terms used in AI work
- Learn enough Python to follow examples
- Learn how to work with data
- Learn how to test and improve results

Then move into:
- Prompt engineering
- Model use cases
- Simple deployment ideas
- Portfolio writing
- Job applications

## 💡 Tips for success
- Work in short sessions
- Focus on one path first
- Finish small projects before starting new ones
- Keep your notes in one place
- Build work that shows your thinking
- Use plain language when you explain your projects
